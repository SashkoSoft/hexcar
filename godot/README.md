# HexCar 3D (Godot)

3D-порт игры HexCar (см. корневой `../README.md`) на **Godot 4.7**, вид сверху.
Вся игровая логика перенесена 1:1 из `../game.html`; рендер — настоящий 3D.

## Запуск

Из корня репозитория:

```bash
./Godot_v4.7-stable_win64.exe --path godot
```

Или открыть папку `godot/` в редакторе Godot 4.7 и нажать ▶ (главная сцена — `Main.tscn`).

## Сборка standalone .exe

Нужны шаблоны экспорта 4.7 (один раз: в редакторе *Проект → Экспорт → Управление шаблонами → Скачать*).
Затем из корня репозитория:

```bash
./Godot_v4.7-stable_win64_console.exe --headless --path godot \
  --export-release "Windows Desktop" build/HexCar3D.exe
```

Получается один самодостаточный файл `godot/build/HexCar3D.exe` (контент встроен).
Заставка Godot отключена (`boot_splash/show_image=false` в `project.godot`).

## Что внутри

- **`Main.gd`** — весь проект в одном скрипте: геометрия плиток Труше, три дороги-дуги
  на плитку, движение машинок по дугам, переходы между плитками, тороидальные порталы,
  вращение ±60°, три режима, очки, а также процедурная сборка 3D-сцены.
- **`Main.tscn`** — корневая сцена (`Node3D` со скриптом).

## 3D-особенности

- Дороги — настоящие 3D-ленты; на пересечении внутри плитки верхняя дорога физически
  приподнимается — получается **эстакада** (машинка проезжает под мостом). На стыках
  плиток высота = 0, поэтому сеть стыкуется без разрывов.
- Плитки вращаются плавной анимацией; декор (деревья, кусты, цветы, камни) крутится вместе с плиткой.
- Направленный свет с тенями, процедурное небо, наклонная камера сверху.

## Управление

Наведи курсор на плитку и поверни её на ±60°:
- стрелки `←` / `→`
- клавиши `A` / `S`
- колесо мыши
- зажми ЛКМ и тяни вбок

## Режимы

**Кольца** · **Погоня** · **Бандит** — как в оригинале (выбор в меню).
