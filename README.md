# HexCar

Демо на чистом HTML5 Canvas: машинка едет сама по дорожкам гексагональных плиток (паттерн в духе плитки Труше). Плитки можно вращать на ±60°, машинка перестраивает маршрут на лету; на замкнутом треке появляется второй авто, при столкновении — авария.

**Играть:** открой `index.html` (или включи GitHub Pages — игра будет по `https://<логин>.github.io/hexcar/`).

## Управление
- Наведи курсор на плитку и крути её на ±60°:
  - стрелки `←` / `→`
  - клавиши `A` / `S`
  - колесо мыши
  - зажми ЛКМ и тяни вбок
- Машинка едет сама по центрам дорог, на пересечениях проезжает «под мостом».

## Файлы
- `index.html` / `game.html` — игра (один самодостаточный файл, без внешних зависимостей).
- `gen_hextile.js` — генератор SVG-плитки и поля (Node), исходная геометрия дорог.
- `hex_tile.*`, `hex_field_5x5.*`, `hex_field_10x10.png` — отрисованные плитка и поля.
