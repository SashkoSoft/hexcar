# HexCar

Игра на чистом HTML5 Canvas (один файл, без зависимостей). Поле собрано из гексагональных плиток с дорогами (паттерн в духе плитки Труше). Машинки едут сами по центрам дорог; на пересечении дорог одной плитки они проезжают «под мостом» (с тенью). Плитки можно **вращать на ±60°**, и машинки на лету перестраивают маршрут — в этом вся игровая механика.

**Играть:** [sashkosoft.github.io/hexcar](https://sashkosoft.github.io/hexcar/) — или открыть `index.html` локально.

## Режимы (выбор в меню)
- **Кольца** — крути плитки и загоняй машинки в замкнутые кольца. Очки = сумма по машинкам `(число задействованных ячеек)²`; круг распался — очки уходят. Замкнутый трек подсвечивается цветом машинки; на новом кольце появляется ещё одна машинка. Столкновение двух машинок — авария, игра сначала.
- **Погоня** — управляя плитками, своей полицейской (быстрее) перехвати бандита.
- **Бандит** — уходи от полиции (она быстрее) как можно дольше.

В режимах гонки поле **тороидальное**: уехал за край — появился с противоположной стороны. Скорость машинок со временем растёт.

## Управление
Наведи курсор на плитку и поверни её на ±60° любым из способов:
- стрелки `←` / `→`
- клавиши `A` / `S`
- колесо мыши
- зажми ЛКМ и тяни вбок

Поворот плитки, на которой едет машинка, притормаживает её и сбрасывает набранную скорость; машинка остаётся на своей (повернувшейся) дороге.

## Оформление
Трава-фон с травинками/камешками, по краям дорог процедурно расставлены камешки, цветочки, кустики и деревца (по удалённости от дороги), вращаются вместе с плиткой. Дорога — светлый асфальт со сплошными краевыми линиями и пунктиром по центру; мост даёт мягкую тень на нижнюю дорогу и машинку под ним.

## Файлы
- `index.html` / `game.html` — игра (идентичны; `index.html` для GitHub Pages).
- `gen_hextile.js` — Node-генератор SVG плитки/поля (исходная геометрия дорог).
- `hex_tile.*`, `hex_field_5x5.*`, `hex_field_10x10.png` — отрисованные плитка и поля.
